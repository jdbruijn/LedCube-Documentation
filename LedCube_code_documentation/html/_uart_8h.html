#define UART1_BRG_CLOCKS   4

Baudrate clock prescaler for UART 1.

This value should be either 4 or 16. 4 = BRG generates 4 clocks per bit period (4x baud clock, High-Speed mode) 16 = BRG generates 16 clocks per bit period (16x baud clock, Standard Speed mode)

UARTx Baud Rate error.

This value is in tenth, so a value of 21 means that the Baud Rate error equals 2.1%.
