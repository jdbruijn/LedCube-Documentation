void Uart1_init ( void  )

Initialize the UARTx module with the given baud rate.

Note
If debug is enabled this function should be called before initializing all other initialize functions. But obviously I/O ports should be configured fist.
Make sure the analog functionality of the RX pin, if it has an analog functionality, is disabled.

void Uart1_putNum ( int32_t  _num,
uint8_t  _base,
bool  _printBase 
)

Send an integer number using the UARTx module.

Note
The number must be within range of a 32-bit integer. Thus the lowest number is -2147483647 and the highest number is 2147483647.
Parameters
_numNumber to send using UARTx.
_baseThe numeral system to use. 
* | _base  | Description                                                      |
* | ------ | ---------------------------------------------------------------- |
* | 2      | Binary (adds suffix 'b' to the output)                           |
* | 10     | Decimal                                                          |
* | 16     | hexadecimal (adds suffix 'h' to the output)                      |
*
_printBaseBoolean value whether the base ('b' or 'h' suffix) needs to be send.
