Detailed Description

Provides Cube- and CubeControl data structures and functions.

Declares and controls the data of the whole LED cube using several functions.

void CubeControlData_switchCubeData ( pCubeControlData_t const  _pCubeControlData)

Switch the CubeData read- and write pointers.

Note
The switch is performed with interrupts disabled using the INTERRUPTS_DISSABLE_AND_SAVE_CPU_IPL and INTERRUPTS_RESTORE_CPU_IPL macros because the CubeData read takes place in an interrupt service routine.
