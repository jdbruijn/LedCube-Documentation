\subsection{Detailed Description}
Control macros for all I/\+O operations. 

When first configuring all ports, the macro \hyperlink{_ports_8h_abd6cc207af38aaeeb3724bfcaac96e66}{P\+O\+R\+T\+\_\+\+I\+N\+I\+T\+I\+A\+L\+I\+Z\+E\+\_\+\+A\+L\+L\+\_\+\+P\+I\+N\+S} should be called before configuring the I/\+O ports. This will to configure all pins in a default state. \begin{DoxyNote}{Note}
This file uses the defines for the ports that are defined in the device\textquotesingle{}s header file. 
\end{DoxyNote}

Reset all the settings of the selected pin. 

This clears the output of the pin and makes it a normal digital output. This is performed by setting and clearing the bits for this pin in the L\+A\+Tx, O\+D\+Cx and T\+R\+I\+Sx registers. The P\+O\+R\+Tx register does not need to be cleared because this is the register for the data on an I/\+O pin, so it is only used for reading the pin\textquotesingle{}s state.

\begin{DoxyNote}{Note}
This does not change the analog functionality of the pin. If it used to be an analog pin the user still need to disable the analog functionality for this specific pin. 

P\+P\+S pins can also be called by their R\+Pn name. For example\+: \hyperlink{_ports_8h_ac9f448d2fe5dfd2551d6390c9da38bf4}{P\+O\+R\+T\+\_\+\+R\+P16} instead of \hyperlink{_ports_8h_a2223ab3e7cfa1b8486e8c5289863d67f}{P\+O\+R\+T\+\_\+\+R\+C0}. 
\end{DoxyNote}
