Detailed Description

Structure with Cube- and CubeControl data.

This contains two RowData_t structure arrays, pointers to the current read and write structure arrays, masks for the BAM rounds and the shift needed for each BAM round.

Field Documentation

Constant array of type BamRoundMasks_t that is used as a BAM mask for each BAM round.

uint8_t BamRoundShift[CUBEDATA_N_BAM_BITS]

Constant array containing the shift needed for each BAM round

RowData_t CubeData0[CUBEDATA_MAX_Z_C][CUBEDATA_MAX_X_C]

First CubeData structure array where the LED data of the whole LedCube is stored in. So this is the data for all 512 LEDs and the CUBEDATA_N_BAM_BITS BAM rounds.

RowData_t CubeData1[CUBEDATA_MAX_Z_C][CUBEDATA_MAX_X_C]

Second CubeData structure array where the LED data of the whole LedCube is stored in. For more info see CubeData0.

pCubeData_t pCubeDataRead

Pointer to the current CubeData structure array where is read from. This can point to either CubeData0 using pCubeData0Base or to CubeData1 using pCubeData0Base.

pCubeData_t pCubeDataWrite

Pointer to the current CubeData structure array where is written to. This can point to either CubeData0 using pCubeData0Base or to CubeData1 using pCubeData0Base.
