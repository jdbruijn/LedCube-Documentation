Detailed Description

Structure of the RGB values of a single row of LEDs.

This contains 32-bit red, green and blue values where the data of a single row with CUBEDATA_MAX_Y_C LEDs in it. These 32-bit values contains the state of the LEDs for the CUBEDATA_N_BAM_BITS BAM rounds and are classified as follows.

Note
The uint32_t value is in binary format and spans all the columns in that row.

BAM round 3 2 1 0
uint32_t value (binary) 00000000 00000000 00000000 00000000
Column (y-value) 76543210 76543210 76543210 76543210
