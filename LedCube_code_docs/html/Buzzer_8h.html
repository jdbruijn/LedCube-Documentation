void Buzzer_set ( uint16_t  _frequency,
uint32_t  _time 
)

Make a sound with adjustable frequency and duration time.

The sound is created using a PWM-signal with a duty cycle of fifty percent.

Note
_frequency must be between BUZZER_MIN_FREQUENCY and BUZZER_MAX_FREQUENCY, and _time must be between BUZZER_MIN_TIME and BUZZER_MAX_TIME.
The output frequency has a maximum deviation of 1.1%.
Parameters
_frequencyFrequency of the sound in Hz. Ranging from BUZZER_MIN_FREQUENCY to BUZZER_MAX_FREQUENCY.
_timeDuration of the sound in ms. Ranging from BUZZER_MIN_TIME to BUZZER_MAX_TIME.
