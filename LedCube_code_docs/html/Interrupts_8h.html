#define INTERRUPTS_DISSABLE_AND_SAVE_CPU_IPL
Value:
uint16_t cpuIntPrio = 0; \
SET_AND_SAVE_CPU_IPL(cpuIntPrio, 7)

Disable interrupts.

Disable interrupts by saving the current CPU interrupt priority level to a variable and setting the CPU interrupt priority level to 7 (the highest level). Using the predefined SET_AND_SAVE_CPU_IPL macro for setting and saving the CPU interrupt priority level.

#define INTERRUPTS_RESTORE_CPU_IPL   RESTORE_CPU_IPL(cpuIntPrio)

Restore the previous CPU interrupt priority level from the saved variable.

Using the predefined RESTORE_CPU_IPL macro for restoring the CPU interrupt priority level. This doesn't necessary mean that interrupts are enabled after this macro, because the CPU interrupt priority level is only restored to it's original value. So if it original value was 7 (interrupts disabled) it will be restored to the same value, resulting in an interrupts disabled situation. The macro INTERRUPTS_DISSABLE_AND_SAVE_CPU_IPL must be in the same scope as this macro.
